# Proxy 24

一个简单的代理插件，白名单很干净，没有任何后门，干净又卫生

## 支持的浏览器

- Google Chrome
- Microsoft Edge
- Firefox
- Quark

## 支持的协议

- HTTP

- HTTPS

- SOCKS4

- SOCKS5

## 应用商店

Firefox-https://addons.mozilla.org/zh-CN/firefox/addon/proxy24/

Microsoft Edge-https://microsoftedge.microsoft.com/addons/detail/proxy24/ehlobeopbpgminfelkofbcmkdigibnej

Quark-已申请(待通过)
